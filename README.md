# Salary Prediction ML Project

This project predicts the salary of a candidate using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn

## Dataset Features
- Experience
- TestScore
- InterviewScore

## Model Used
Linear Regression

## Steps in Project
1. Load dataset using Pandas
2. Data analysis using info() and describe()
3. Feature selection
4. Train-test split
5. Train Linear Regression model
6. Evaluate model using RMSE and R2 score
7. Predict salary for new candidate

## Output
The model predicts salary based on experience, test score and interview score.
